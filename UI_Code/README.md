# Musicguru-app

## Project setup instructions
```
git clone git@github.com:bhuvanabasapur/TeamInvinsibles.git
cd UI_Code
pushd UI_Code
yarn install
popd
```

### Compilation
```
yarn serve
OR
npm run serve
```
